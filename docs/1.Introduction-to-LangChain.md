Instead of hardcoding, you define placeholders (`{variable}`) that can be **filled at runtime** with different inputs.

## ðŸ”¹ Example: Using PromptTemplate
```python
from langchain.prompts import PromptTemplate

# Define template
template = "Translate the following text into {language}:\n\n{text}"

# Create PromptTemplate
prompt = PromptTemplate(
    input_variables=["language", "text"],
    template=template,
)

# Format with dynamic inputs
final_prompt = prompt.format(language="French", text="Hello, how are you?")
print(final_prompt)
